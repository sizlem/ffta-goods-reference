# FFTA goods reference

일본 여행 중 매장 직원에게 보여주기 위한 `Final Fantasy Tactics Advance` 굿즈 참고 자료입니다.

## Main file

- `show_to_shop.html`: 매장에 바로 보여줄 수 있는 3개 언어 자료
- 하단 플로팅 버튼으로 `日本語 / 한국어 / English` 전환
- 각 상품은 접히는 상세 섹션으로 구성
- 사진은 클릭/터치하면 확대, 다시 누르거나 ESC를 누르면 닫힘
